Este programa dibuja una mariposa, junto con un escenario, en teoría al momento de ejecutarlo no debiese ocurrir ningún problema
El programa dibuja una mariposa con 4 triángulos por alas, rotados de cierta manera para asimilarse a la forma de una mariposa,
luego el programa se encarga de variar el tamaño de las alas para dar la sensación de movimiento, para por último en las últimas líneas
implementar el traslado de la mariposa, el cual fue implementado antes que el aleteo. Parte del contenido fue ajustado al aspect ratio 16:9,
pero no fue necesario ajustar todo el contenido, ya que hice muchas partes al "ojo". (Me di cuenta cuando hice el sol)

Luego está la función del escenario, la cual retorna únicamente el sol y el pasto, esto se debe a que las flores fueron dibujadas dentro de la misma
función en un ciclo for. 

Los adicionales antes comentados serían el sol, el pasto, las flores y por último las antenas.
